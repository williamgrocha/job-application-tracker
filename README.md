# Job Application Tracker

A web application designed to help users organize and track their job applications. The system allows users to manage job opportunities, monitor application statuses, and keep track of important deadlines in a one place.

This project was built to practice backend development concepts such as web application architecture, database integration, and RESTful design using Python and Flask.

---

## Features

- Create and manage job applications
- Track application status (Applied, Interview, Offer, Rejected)
- Store deadlines and notes for each application
- View and update existing applications
- Persistent data storage using SQLite
